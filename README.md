## описание ключевых функций
+ double function(double x)  
Назначение: Вычисление значения функции по заданным математическим формулам в зависимости от интервала  
Логика:  
При x < 2: (atan(x) - sin(x)) / x³  
При 2 ≤ x < 9: e^(cos(x)) / ln(3+x)  
При x ≥ 9: sin(2+x) / (1 + x⁶)  
+ void build_tabl(float start, float step, int quantity, char* filename)  
Назначение: Генерация таблицы значений функции с записью в файл  
Работа:  
Форматированный вывод в консоль и файл в виде таблицы  
Автоматическое открытие/закрытие файла с проверкой ошибок
+ char* monotony(double start, double end, int points, char* filename)
Назначение: Анализ монотонности функции на интервале
Алгоритм:  
Разбивает интервал на равные отрезки  
Сравнивает значения в соседних точках  
Определяет: возрастание/убывание/постоянство/немонотонность  
Сохраняет вывод в файл и возвращает текстовое описание
+ double X_on_Y(double targetY, double precision, double startX, double endX)  
Назначение: Поиск аргумента x, при котором function(x) ≈ Y  
Метод:  
Проходит по интервалу с малым шагом (0.00001)  
Находит первую точку, где разница меньше погрешности  
Возвращает найденный x или 0 при отсутствии решения
+ double derivative(double x, double precision)  
Назначение: Численное вычисление производной функции в точке  
Формула: [f(x+h) - f(x)] / h где h = погрешность
+ int write_doc(char* filename, double value, int mode)
Назначение: Универсальная запись численных результатов в файл  
Режимы:  
mode=1: запись значения функции в точке  
mode=4: запись найденного x по y  
mode=5: запись значения производной
+ void read_and_build_table(char* filename)
Назначение: Чтение числовых данных из файла и построение таблицы значений  
Алгоритм:  
Вычисление function(x) для каждого прочитанного x  
Форматированный вывод таблицы в консоль и добавление в конец файла
